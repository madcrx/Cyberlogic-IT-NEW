PROBLEM: .htaccess not visible after extraction
SOLUTION:
- Open File Explorer
- Click "View" tab
- Check "Hidden items"
- Check "File name extensions"
- .htaccess should now be visible

PROBLEM: FileZilla won't connect
SOLUTION:
- Check FTP credentials from hosting provider
- Try port 21 (FTP) or port 22 (SFTP)
- Disable Windows Firewall temporarily
- Contact hosting support

PROBLEM: Website shows 403 Forbidden
SOLUTION:
- Check file permissions (644 for files, 755 for folders)
- Verify index.html is in root directory
- Contact hosting provider

PROBLEM: Admin panel not accessible
SOLUTION:
- Clear browser cache (Ctrl + Shift + Delete)
- Try different browser
- Check admin/ folder uploaded correctly
- Verify folder permissions

PROBLEM: Contact form not working
SOLUTION:
- Check PHP version (needs PHP 7.0+)
- Verify FormSubmit.co verification
- Test with different email address
- Check spam folder
